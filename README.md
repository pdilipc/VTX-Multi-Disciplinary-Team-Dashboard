# VTX - Multidisciplinary Team Dashboard

Verada VTX — MDT Dashboard (MVP, in development). A FHIR R4 clinician dashboard for multidisciplinary neurorehab: patient roster, readiness review, session planning and structured FHIR write-back, with neuro-specific panels (Fugl-Meyer, PHQ-9/GAD-7, Dozee monitoring) using standard LOINC codes. Stack: React + Vite + TypeScript, a FHIR service layer, Supabase persistence. Runs against a FHIR R4 server with synthetic data only. Corresponds to project-plan tasks S2.1–S2.3; early-stage (TRL 3). Not a dependency for Year-1 recoveriX clinical delivery.
